설치 가이드

5단계 설치

1
APK 다운로드
위 다운로드 버튼 클릭 → 폰 다운로드 폴더에 저장 (시드 가입 신청 후 활성화)
2
알 수 없는 출처 허용
설정 → 보안 → "알 수 없는 출처에서 앱 설치 허용" → 파일 관리자에 권한 부여
3
APK 탭 + 설치
파일 관리자에서 app-dev-debug.apk 탭 → "설치" → 확인 (앱 ID: com.uncounted.app.dev)
4
앱 실행 + Google 로그인
Uncounted Dev 앱 아이콘 탭 → Google 계정 (Gmail 등) 로그인 → 본인 동의
5
권한 부여 + 자동 스캔
통화 녹음 + 통화 기록 + 마이크 권한 부여 → 자동 스캔 + 메타 식별 시작
개인정보보호법 제15조 + 통신비밀보호법 + PIPA 30조 11항 (음성=생체정보) 정합.
양자 동의 완료 후에만 회사 서버에 전송 + 처리.
