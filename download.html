이 앱은 서비스 초기 단계 한정으로 직접 배포해 드립니다 — 구글 플레이 정식 등록은 추후 진행될 예정입니다. 설치하실 때 "알 수 없는 출처에서 앱 설치 허용"을 켜셔야 합니다. 화면에 나오는 보안 경고는 정식 인증을 받지 않은 단계라서 나오는 것이며, 앱 자체는 한국인터넷진흥원(KISA) 지침과 「개인정보보호법」을 모두 따릅니다. 안심이 되지 않으시면 구글 플레이 정식 등록 후 받으셔도 좋습니다.
설치 가이드

5단계 설치

1
앱 파일 받기
위 받기 버튼을 누르시면 휴대폰의 다운로드 폴더에 저장됩니다 (가입 신청 후 활성화).
2
"알 수 없는 출처" 허용
휴대폰 [설정] → [보안] → "알 수 없는 출처에서 앱 설치 허용"을 켜시고, 파일 관리자 앱에 권한을 주세요.
3
앱 파일을 눌러서 설치
파일 관리자에서 app-dev-debug.apk 파일을 누르시면 → "설치"를 누르시고 → 확인을 눌러 주세요.
4
앱 실행 + 구글 로그인
UNCOUNTED 앱 아이콘을 누르신 뒤 → 구글 계정(지메일 등)으로 로그인 → 본인 동의를 눌러 주세요.
5
권한 허용 + 자동 확인 시작
통화 녹음, 통화 기록, 마이크 권한을 모두 허용해 주시면 → 휴대폰의 통화를 자동으로 확인하기 시작합니다.
「개인정보보호법」 제15조, 「통신비밀보호법」, 「개인정보보호법」 음성 생체정보 보호 조항을 따릅니다.
본인과 상대방이 모두 동의하신 뒤에만 회사 서버로 전송하여 처리합니다.
